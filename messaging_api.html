Messaging API

Freshmaker also sends AMQP or Fedmsg messages when events or builds change its state.

event.state.changed

This message is sent on every Freshmaker Event’s state change. The message contains Event JSON representation as defined in API version 1.

event.state.changed.min

This message is sent on every Freshmaker Event’s state change. The message contains Event JSON representation as defined in API version 2.

build.state.changed

This message is sent on every Artifact Build’s state change. The message contains Artifact Build.
